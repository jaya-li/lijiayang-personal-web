## 工作经历

### TikTok UG｜AI 设计 Owner（2025.4 – 至今）

**项目介绍：** 围绕 TikTok 用户增长目标，提升增长活动创意供给效率与投放效果，推动 AIGC 从「辅助设计」升级为可规模化复用的增长基础设施。

**核心挑战：** 用户增长场景下，创意生产链路长、人工依赖高、迭代效率低，不同端（Lite / Main / Pro）与不同活动的素材需求分散，导致优质创意难以稳定、快速地大规模供给。

**工作内容与结果要点：**

- 以增长提效为目标，系统梳理视觉设计生产流程，识别可由 AI 接管的关键节点，推动 Agent 化改造与规模化落地；将玩法能力抽象为可复用的增长 Agent，把原本 3 天的人工流程压缩至 1.5 小时，整体提效 97.2%，AI 介入率提升至 95%+，并在换肤活动中完成可用性验证。
- 为提升 TikTok 各端 ASO 推广图投放量与产运设计协作生图质量，从 0 到 1 作为 owner 搭建 **Groot AI Agent** 平台，打通模型调度、Prompt 策略、交互流程、改图链路与审核联动机制，形成「对话式生成—审核—发布」的闭环；将 ASO 推广图生产时长缩短至 30 分钟，提效 98%，平台累计产出素材带来获客 1300 万+、促活 700 万+。
- 面向日韩本地化增长需求，针对通用模型在区域审美与风格表达上不稳定的问题，基于 LoRA 训练拓展 10+ 种日韩风格模型，支撑 20+ UG 活动玩法。
- 为解决增长素材「只生产、不沉淀」的问题，设计 **数据智荐 Agent**，探索「历史素材 + Prompt + 投放数据」的回流机制，为推荐优化与策略复用建立数据基础。

### Seed 文生图｜Seed Prompt Engineering（2024.4 – 2024.11）

**项目目标：** 提升 Seed 文生图模型在真实业务场景中的美感、图文匹配和风格稳定性，建立可复用的 Prompt 评估与优化体系。

**核心挑战：** 文生图模型在摄影、设计、三维、插画、字体、艺术及超写实等场景中，存在审美表达不稳定、Prompt 依赖个人经验、优化过程难沉淀复用等问题。

**工作内容与结果要点：**

- 基于 PE 机制设计通用 Prompt 模板，抽象长期可复用的结构化生成范式，推动 Seed 文生图大模型生成美感提升 120%。
- 建立风格效果、美感词与冗余词改写规则，推动 Prompt 优化从「经验试错」走向规则化、标准化。
- 设计「美学词立体维度」PE 方法，提升不同层次美学词的组合与调度能力。
- 在人工评审基础上引入 **LLM-as-a-Judge** 辅助评测，用于图文匹配、主体完整性、风格偏差等维度的结构化判断，提升样例筛选、问题归因与 Prompt 迭代效率。
